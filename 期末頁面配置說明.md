## css設計規範，例如哪些元素只能多高、內距、外距、配色等...
- 1.整體風格，使用 深色系漸層側邊欄：如 linear-gradient(135deg, #2c3e50, #34495e)，配色偏向:藍灰、青藍、金棕點綴、薄荷綠強調色，主色：#2c3e50、#34495e，強調色：#46C2AC、#61FAE8，次要點綴：#D6AE72，內容文字：#333、#666、#3835B0
- 2.左側導覽列（Sidebar）:固定在左側、寬度 250px，可折疊成 60px，平滑動畫（0.3s）切換寬度，內容採垂直排列，hover 有透明白背景效果，支援「active」項目的背景高亮效果，左側導覽列顏色不得影響內文，平鋪從網站最上面到網站頁面最底下層，{左側導覽列}會在{最上方標題區塊}和{頁腳}和內文三項上面，所以左側導覽列從頭至尾都不會動，所有可向外連結html的按鈕大小需要平均。導覽列內字體顏色為文字大小 1.25rem，導覽列內的字體設計程式碼如下:<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Hachi+Maru+Pop&display=swap" rel="stylesheet">

- 3.導覽列內各連結至其他html的連結字體前的圖案，大小一公分需要一致，圖案來源於images資料夾內的資訊icon資料夾，首頁連結字前的連結圖案是首頁.jpg，前菜連結字前的連結圖案是前菜.jpg，沙拉連結字前的連結圖案是沙拉.jpg，湯品連結字前的連結圖案是湯品.jpg，依此類推下去。左側導覽列背景請使用深色質感漸層，以藍灰色為基底，從左上到右下呈現 #2c3e50 → #34495e 的 135 度漸層，帶有沉穩、科技感的配色。

- 4.最上方標題區塊:留一個banner區塊放置底圖，底圖圖片來源位置：images/ .jpg，只需要5公分的大小的article區塊，其中包含標題區塊H1以及內文區塊content，article 下的H1區塊文字 大小 1.6rem，字體顏色 #46C2AC；content 區塊文字大小 1.1rem，字體顏色 #3835B0。，Header / Banner，Header 跟隨 sidebar 寬度移動（250px → 60px），Banner 使用滿版圖片 object-fit: cover，右上角配置搜尋／選單等控件，以白色半透明背景按鈕呈現

- 5.Main 主內容區:主內容區會因 sidebar 寬度改變 margin-left，Title 使用 #262A66、內文使用 #3835B0，內容最大寬度約 800px，清楚易讀

- 6.article區塊中包含標題區塊H1以及內文區塊content，article 下的H1區塊文字 大小 1.6rem，content 區塊文字大小 1.1rem 

- 7.產品卡片區塊（Cards）:三欄式彈性排版 (flex + wrap)，卡片具圓角 15px、陰影、hover 位移效果，價格使用亮色強調 #2B4969，保持整齊，圖片和價格之間有一條虛線間隔，產品名稱的區塊以 H2 命名，section1 下的產品名稱H2 區塊，每一個區塊中間都有一條0.5公分空隙，文字大小 1.25rem，文字顏色 #D6AE72。價格的文字大小 1.25rem，文字顏色 #61FAE8。 section1每一列最多三個section1，超過就換行。

- 8.RWD 響應式:小於 768px  時：卡片改為單欄，sidebar 預設縮小成 60px。header / main / footer 自動調整 margin-left

- 9.Footer:深色背景 #2c3e50，左右分佈對齊 

- 10.網站最右上角需要有一個小長方形邊角圓角不是直角，放音樂背景來源於media資料夾內的音訊，該圖案在有任何人點開網站時為黑色暫停鍵，有人按暫停後為白色撥放鍵，同時自動縮成小正方形
